# C# Modules

Welcome to the C# Modules! This curriculum is designed to help you learn C# through practical exercises, inspired by the 42 School C++ Modules.

## Getting Started

1.  Navigate to `Module_00` to start your journey.
2.  Read the `subject.md` file in each module directory for instructions.
3.  Each exercise is located in its own directory (e.g., `ex00`, `ex01`).
4.  Use `dotnet run` inside an exercise directory to run your code.

## Prerequisites

- .NET 8.0 SDK (or newer)
- A text editor (VS Code recommended)
